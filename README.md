# SqlRefactorAnalyzerEXE

Compiled Windows binaries (releases) for [SqlRefactorAnalyzer](https://github.com/JankeUwe/SqlRefactorAnalyzer) —
a native C#/WinForms desktop tool for T-SQL refactoring and query-plan analysis.

This repo contains only build output (exe + dependencies), not source code.
See the source repo for documentation, build instructions, and issue tracking.

## Releases

Download the latest zip from the [Releases](../../releases) page, extract, and run `SqlRefactorAnalyzer.exe`.
Requires .NET Framework 4.8 (Windows).
